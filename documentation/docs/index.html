Openfire Connection Manager Module

Connection managers concentrate XMPP client connections and forward their traffic to an XMPP server.

Documentation:

Developer Documentation:

  • Building the Source - Instructions for downloading and compiling the connection manager source code.
  • Translator Guide - Information for those interested in translating the text in the connection manager module into other languages.
